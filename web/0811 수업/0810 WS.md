# 문제 4. 선택자

```html
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Document</title>
    <style>
        #ssafy > p: nth-child(2) {
            color: red;
        }
    </style>
</head>
<body>
    
    </body>
```

nth-child() 는 ssafy 하위 아래 n번째가 p인지 아닌지 비교 -> p가 아니면 실행 안됨,





```html
<style>
  #ssafy > p:nth-of-type(2){
    color: blue
    }
      /style>
```



nth-of-child() 는 ssafy 하위 아래 n번째 p에 적용.
