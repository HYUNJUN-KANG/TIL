# 문제 4. CSS 개념

```html
1) HTML와 CSS는 각자 문법을 갖는 별개의 언어이다.
2) 웹 브라우저는 내장 기본 스타일이 있어 CSS가 없어도 작동함.
3) 자식 요소 프로퍼티는 부모의 프로퍼티를 모두 상속받진 않는다. (block관련 요소)
4) 디바이스 마다 화면의 크기는 viewport가 담당. viewport를 고려한 단어인, vw, vh, vmin 을 사용 -> %가 아님.
5) id 값은 중복하면 안된다.
```
